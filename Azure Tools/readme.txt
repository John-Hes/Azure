Listing helpful Azure tools. I come across helpful code or apps others have created and would like to track them here.

1.  Draw.io
      https://draw.io
      Used to create diagrams

2.  Azure Resource Inventory (ARI)
      https://github.com/microsoft/ARI?search=1
      Generates a nice report for a detailed inventory of all Azure resources
      Can run in Cloud Shell

3.  Azure Quick Review (AZQR)
      https://github.com/Azure/azqr/blob/main/README.md
      Analyzes Azure resources to ensure compliance with Azure's best practices and recommendations
      Can run in Cloud Shell

4.  Rubrik's Azure Sizing Script
      https://github.com/rubrikinc/rubrik-sizing-scripts/tree/master/CLOUD
      Rubrik has a great script to pull sizing information across your whole Azure environment
      Can run in Cloud Shell

5.  Export Role Assignments scripts
      https://github.com/MortenPedholt/Powershell/blob/master/Azure/RBAC%20Roles/Export-RoleAssignments.ps1
      Great script to review permissions assigned across your Azure environment
